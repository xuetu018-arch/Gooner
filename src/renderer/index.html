设为 0:0:0 时不会自动消失；大于 0 时，图片和 AI 文本弹窗会在到时后缓慢淡出并关闭（不影响视频）。

开启后，视频将从随机进度开始播放（避开最后几秒）。且视频受“自然消失时间”控制。若未到消失时间但播放完毕，也会自动关闭。开启此功能将覆盖“视频播放完成自动关闭”。

开启后无法通过按钮、点击或 Esc 关闭弹窗。仅在自然消失时间 > 0 时生效，主面板关闭不受影响。

开启后将隐藏主界面上的文件夹路径和 API 密钥等敏感信息。同时允许你在弹出的图片/视频上点击右键，在文件管理器中定位该文件。
